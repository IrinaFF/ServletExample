структура проекта:
src/java/servlets — тут будут лежать сервлеты
src/resources — тут будут лежать все ресурсные файлы которые буду нужны для проекта.
src/webapp — тут лежат все файлы которые будут использоваться во view (к примеру img, css)

Servlet — это Java-интерфейс, реализация которого расширяет функциональные возможности сервера.
Servlet взаимодействует с клиентами посредством принципа запрос-ответ за частую это запросы GET, POST по протоколу HTTP/HTTPS.
Для создания сервлета нам нужно подключить зависимости в Maven
